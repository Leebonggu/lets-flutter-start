## Dart

### const vs final

- 둘다 값을 대입하면, 바꿀 수 없다.

-final은 실행중에 값이 결정됩니다.
-final은 이 파일이 실행될 때 해당 위치에서 값이 결정됩니다.

-const는 컴파일시에 값이 결정됩니다.
-const는 이 파일을 컴파일할 때 기계어로 번역될 때 값이 결정됩니다.
